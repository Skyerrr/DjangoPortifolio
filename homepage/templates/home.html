My name is Felipe Gimenes. I'm a Full-Stack developer based in Rio de Janeiro, Brazil. I develop fully functional websites, web scraping projects, and a macro software called SkyMacro.

BillyMods

BillyMods is a multi-language website made in Flask and SQL Database. You can add, edit or remove categories, subcategories and products. All changes can be made by the Admin login inside the website and the products are fully customizable.

SkyMacro

SkyMacro is the solution for games and applications that have macro protection. Create,edit and set a hotkey for your macro in the UI or edit manually in the .txt file.

ZiScrap

ZiScrap is a simple web scrapping project made with BeautifulSoup and Selenium that scraps Address, price and link from Zillow's website chosen location and automatically put in Google Sheets.

BlogExample

Basic blog made with Flask, SQL and Bootstrap. Secure register and login with WTForms. Posts and comments are saved in SQL Database.

Amazon Scraping Project

Amazon Web Scraping Project with BeautifulSoup. Define your search product in "run.py", The program will save the html file then extract name and price to a csv file with the search product name.
